# Verifica pratica: Git, Shell, Docker

## Obiettivo

In questa prova dovrai:

1. clonare il repository;
2. creare una tua cartella di lavoro;
3. realizzare un semplice script Python;
4. creare un `Dockerfile` per eseguire lo script in un container;
5. scrivere uno script shell per automatizzare build ed esecuzione;
6. generare un file di output;
7. salvare tutto con `git add`, `git commit` e `git push`.

---

## Requisiti

Prima di iniziare assicurati di avere disponibili:

- Git
- Docker
- Python 3
- una connessione Internet attiva (per scaricare l'immagine base Docker, se necessario)
